# Anthropic

Anthropic is the AI lab behind Claude and Claude Code; it appears in the podcast through the two people building Claude Code — creator Boris Cherny and engineering leader Fiona Fung, both senior [Meta](meta.md) alumni — plus its interview culture as described by a hiring coach.

## What the episodes reveal

- **Claude Code from the inside.** Boris Cherny's manager Ben Mann pushed him to "build for the model six months from now, not today" — Claude Code was mediocre until Sonnet/Opus 4 landed. Now ~80–90% of Claude Code is written with Claude Code, and per-engineer productivity rose ~70% even as the company tripled; usage extends far beyond code (data scientists run swarms of it, half the sales team uses it) ([Boris Cherny](../sources/boris-cherny-creator-of-claude-code.md)). On rivals like Codex, Cherny doesn't use competitor products and tells the team more competitors is a sign of success ([same](../sources/boris-cherny-creator-of-claude-code.md)).
- **Mission-driven culture.** Fiona Fung wasn't job hunting — using Claude via Meta's internal DevMate convinced her AI was "already here," and Anthropic's safety-first mission-orientation was visible from day-one onboarding, not just a tagline; it pulled her from a VR role she loved. She finds Claude Code's iteration loop (idea → build → internal launch → feedback → public launch) faster than even early Facebook Marketplace, and her top concern is preserving that agility as the team grows ([Fiona Fung](../sources/anthropic-eng-leader-and-ex-senior.md)). Her signature dogfooding practice continues there: "I'm using Claude Code to build Claude Code" ([same](../sources/anthropic-eng-leader-and-ex-senior.md)).
- **Engineering discipline with AI.** Cherny's rule: hold model code to exactly the same bar as human code — vibe coding is for prototypes, pairing-with-plan-mode is the default, and he still hand-writes the core query loop ([Boris Cherny](../sources/boris-cherny-creator-of-claude-code.md)).
- **Interview culture.** Anthropic's "Hold Light and Shade" value wants candidates to articulate both positive and negative implications of AI — when telling an AI project story, explicitly cover how you assessed and mitigated risks ([Meta hiring lead](../sources/meta-hiring-lead-on-behind-the-scenes.md)).
